# ToonFlow Mobile 原型

React Native / Expo JSX 原型，5 个主 Tab 页。

## 页面结构

```
┌──────────────────────────────────────┐
│  ToonFlow Mobile                     │
├──────────────────────────────────────┤
│                                      │
│  🏠 首页                              │
│  ├── 欢迎卡片                         │
│  ├── 一句话创作入口                    │
│  ├── 统计卡片 (进行中/已完成/分镜)      │
│  ├── 最近项目列表 (进度条)              │
│  └── 流水线状态 (横向滚动9阶段)         │
│                                      │
│  ✨ 创作                              │
│  ├── 灵感输入 (多行文本框)              │
│  ├── 风格选择器 (横向滚动chips)         │
│  ├── 集数选择 (3/5/8/10)              │
│  ├── 题材类型                         │
│  ├── 🚀 开始创作 按钮                  │
│  └── 实时进度 (SSE 5阶段)              │
│                                      │
│  📁 项目 (详情)                       │
│  ├── 项目头部 (名称/风格/进度)          │
│  ├── 集数 Tab (横向滚动)               │
│  ├── 内容 Tab (剧本/分镜/视频)         │
│  │   ├── 📝 剧本视图                  │
│  │   ├── 🎬 分镜网格 (2列卡片)         │
│  │   └── 📹 视频播放器 + Timeline      │
│  └── 操作按钮                         │
│                                      │
│  🎨 资产库                            │
│  ├── 筛选器 (全部/角色/场景/道具)       │
│  └── 资产网格 (2列卡片+头像)           │
│                                      │
│  👤 我的                              │
│  ├── 用户卡片                         │
│  ├── 统计 (项目/分镜/视频)             │
│  └── 菜单 (配置/成本/风格/安全/统计)    │
│                                      │
├──────────────────────────────────────┤
│  🏠    ✨    📁    🎨    👤           │
│  首页  创作  项目  资产  我的          │
└──────────────────────────────────────┘
```

## 设计规范

- **主色**: #8B5CF6 (紫色)
- **背景**: #0D0D12 (深黑)
- **卡片**: #1A1A24 (深灰)
- **成功**: #10B981 (绿)
- **警告**: #F59E0B (橙)
- **字体**: 系统字体，中文优先

## 运行

```bash
# 初始化 Expo 项目
npx create-expo-app toonflow-mobile
cd toonflow-mobile

# 复制原型文件
cp ../mobile-prototype/App.jsx ./App.jsx

# 运行
npx expo start
```

## API 对接

所有 API 调用指向 ToonFlow 后端:
- `POST /other/login` — 登录
- `POST /project/oneClickGenerate` — 一句话生成 (SSE)
- `POST /project/getProject` — 项目列表
- `POST /project/getPipelineState` — 流水线状态
- `POST /assets/getAssets` — 资产列表
- `GET /artStyle/getArtStyle` — 风格列表
